# breakaway

## Requirement

## Setup
To generate IDEA project files:

    ./gradlew idea
    open breakaway.ipr

## Build
To create a executable "fat jar":

    ./gradlew build

## Run the app locally
Start the app:

    ./gradlew bootRun

Set old Java version:

    export JAVA_HOME=`/usr/libexec/java_home -v 1.8
